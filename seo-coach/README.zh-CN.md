## 这是什么

SEO Coach 是 AK（[小红书 @AK SEO Labs](https://xhslink.com/m/3fZJnT4jDHe)）做的一个虚拟陪跑教练 Skill。

它不丢报告、不甩你一份 251 条规则清单、不假装两小时就能帮你解决所有 SEO 问题。

它做的事只有一件：**陪你自己看懂自己的网站**。

用苏格拉底式提问一步步带你找问题、搞清楚原因、决定下一步。自己亲手走一遍，才是真的学会。

---

## 为什么做它

AK 长期在小红书、Threads 公开分享 SEO 教程、拆穿低端服务商的套路。在整理网络上各种公开 SEO 资料的过程中，AK 发现一件事：

**AI Agent 其实非常适合用来学 SEO——但大部分人不知道从哪儿入手，也不知道怎么提对问题。**

与其让新手继续在一堆互相矛盾的信息里兜圈子，不如直接把 AK 觉得真有价值的公开资料和教学逻辑，整合进一个 AI 陪跑教练——让你从第一句话就走在对的方向上。

SEO Coach 就是这么来的。

---

## 你能拿到什么

**从零开始的 SEO 入门陪跑**
带你走完 SEO 新手该懂的基础，包含：
- **技术层**：爬虫抓取、索引收录、HTTPS、Schema、Core Web Vitals
- **页面层**：title、meta、H1、alt、URL 结构
- **内容层**：搜索意图、可读性、E-E-A-T、旧文清理
- **链接生态**：内链架构、外链基础
- **数据层**：GSC 怎么看、GA4 基础、SEO 效果衡量
- **常见场景**：流量掉了、改版、新站、CMS 平台问题、SEO 误区辨别
- **进阶主题**：Topical Map、SERP Features、AI 搜索准备度、电商 SEO、本地 SEO（补充）

一共 18 个 Audit 模块（5 层架构），每个模块一次只盯一个问题，给你一个最小可执行的下一步。不会一次倒整套，不会甩你 251 条清单。

**整合公开 SEO 资源**
内容来自 Google Search Central、Ahrefs 教程、Aleyda Solis、LearningSEO、Lily Ray、Marie Haynes、Whitespark 等主流公开来源。再加上 AK 自己实战总结的补充：服务商鉴别框架、AI 写文章的正确架构、旧文优化优先级、内链规则等。

**持续跟进陪跑**
- 每次 Session 自动存进度（`seo-progress.md`）
- 每个待办事项实时跟踪（`seo-actions.md`）
- 下次回来直接接着做，不用重新交代背景

**直连你的后台数据**
GSC / GA4 / CMS 可以接 MCP / API，教练直接读数据，省得你反复截图贴出来。

**内置版本检查**
直接跟教练说「有没有新版？」就会自动去 GitHub 比对最新版本，告诉你怎么更新。

## 适合谁

- 低竞争本地商家、小型品牌站、服务业网站——基础 SEO 做扎实就能拉开差距
- 想入门 SEO 但不知道从哪儿开始，不想被服务商忽悠
- 已有网站，想知道先修哪儿性价比最高
- 发了几十篇文章但流量一直起不来，搞不清楚问题在哪
- 中高竞争内容站 / 电商——适合当问题排查框架辅助

## 不适合谁

下面这些已经超出免费陪跑能覆盖的范围，需要更深的 pipeline 设计或专业判断：

- 语义 SEO / 主题集群 / 内容权威建设

- 品牌矩阵 / PBN 站群流量 / GEO 优化
- 流量大幅下跌持续数周、改版迁站、大量掉收录
- YMYL 高风险行业（医疗诊断建议、金融投资建议、法律个案意见等需要专业执照判断的内容）

## 怎么开始

把这个 repo 的内容放进你 AI agent 的 skills 文件夹就能用。

**安装步骤**：
1. Clone 这个 repo（或下载 zip）：
   ```
   git clone https://github.com/akseolabs-seo/seo-coach
   ```
2. 把整个文件夹放进你 AI agent 的 skills 目录（路径看你用的平台）
3. 在 AI agent 开一个新对话，直接把你的 SEO 问题或网址扔进去

第一次会问你想用哪种模式：快速问个概念 / 给网址找一个重点问题 / 系统学 SEO 持续跟进。

> 💡 国内访问 GitHub 太卡的话，可以挂梯子或者用 [GitLab 镜像](https://gitlab.com/akseolabs/seo-coach)（同一份内容）。

## 最后说一句

这个 Skill 不会让你变成 SEO 大神。SEO 本来就没标准答案——每个人有自己的玩法，每个网站情况都不一样。

但它能帮你走好第一步：搞清楚自己网站到底卡在哪儿、把第一个关键词排上去、把可以持续做下去的基础打牢。从这儿开始，剩下的靠你自己慢慢攒。
