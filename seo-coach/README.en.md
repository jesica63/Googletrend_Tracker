## What This Is

SEO Coach is a virtual coaching Skill designed and trained by AK ([@darkseoking](https://www.threads.com/@darkseoking)).

It doesn't dump reports. It doesn't hand you a 251-rule checklist. It does one thing: **helps you understand your own website by asking the right questions**.

Using Socratic dialogue, it guides you step-by-step to find problems, understand why they exist, and decide what to fix next — so you actually learn SEO instead of just following someone else's instructions.

---

## Why This Exists

AK (@darkseoking) is a Threads SEO KOL who shares SEO fundamentals openly and calls out low-end vendor tactics. While curating useful public SEO resources, AK noticed something:

**AI Agents are actually a great way to learn SEO — but most people don't know where to start, or how to ask the right questions.**

Instead of letting beginners keep circling in contradictory advice, AK packaged the public teaching logic and curated resources into an AI coaching experience — so your very first question puts you on the right track.

That's SEO Coach.

## What You Get

**A complete SEO beginner curriculum, walked through with you**
Covers what every SEO beginner needs to learn:
- **Technical**: crawlability, indexing, HTTPS, schema, Core Web Vitals
- **On-page**: title, meta, H1, alt, URL structure
- **Content**: search intent, readability, E-E-A-T, content pruning
- **Link ecosystem**: internal link architecture, backlink basics
- **Measurement**: how to read GSC, GA4 basics, SEO performance metrics
- **Common scenarios**: traffic drops, redesigns, new sites, CMS-specific issues, SEO myth-busting
- **Advanced topics**: topical maps, SERP features, AI search readiness, e-commerce SEO, local SEO (supplementary)

18 audit modules across 5 tiers, one focused problem at a time, one executable next step. No firehose, no 251-rule checklist.

**Curated from public SEO resources**
Material drawn from Google Search Central, Ahrefs tutorials, Aleyda Solis, LearningSEO, Lily Ray, Marie Haynes, Whitespark, and other mainstream public sources. Plus AK's own field-tested additions: vendor evaluation frameworks, AI writing architecture, old article optimization, internal link rules.

**Persistent session tracking** — progress saved to `seo-progress.md`, action items tracked in `seo-actions.md`. Come back next week and pick up exactly where you left off.

**Direct data integration** — connect GSC / GA4 / CMS via MCP / API so the coach reads your actual data directly. No more copy-pasting screenshots.

**Built-in update check** — just ask "any updates?" and the coach checks GitHub for the latest version and tells you what's new.

## Getting Started

Drop this repo into your AI agent's skills folder and you're set.

**Installation**:
1. Clone the repo (or download the zip):
   ```
   git clone https://github.com/akseolabs-seo/seo-coach
   ```
2. Place the folder in your AI agent's skills directory (path varies by platform)
3. Start a new conversation in your AI agent and describe your SEO problem or drop a URL

The coach will ask which mode fits: quick concept question / grab the one biggest issue from a URL / systematic SEO learning with persistent tracking.
