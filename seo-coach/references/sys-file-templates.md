# 檔案模板

這個 skill 會在**完整陪跑模式**下，在用戶的資料夾內建立兩個檔案，用來記錄進度和行動清單。輕量模式預設不落盤。

---

---

## 更新規則

- **seo-progress.md**：每次 session 結束時覆寫整個檔案
- **seo-actions.md**：每次 session 結束時更新，保留用戶已打的 [x]，只新增/修改項目，不刪除用戶的記錄
- **seo-actions.md 去重**：同一個問題如果已經存在，就更新原條目，不重複新增；完成後只移到 `✅ 已完成` 一次
