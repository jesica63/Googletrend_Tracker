# SEO 術語表（中英對照）

用戶在對話中碰到不懂的詞時，用這份表格快速解釋。每個詞給一句話的白話說明。

---

## 技術類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| 爬蟲 | Crawler / Spider / Bot | Google 派去讀你網站的機器人 |
| 爬行 | Crawling | 爬蟲造訪並讀取你網站內容的動作 |
| 索引 | Indexing | Google 把你的頁面收錄進搜尋資料庫 |
| 排名 | Ranking | 你的頁面在搜尋結果出現的位置 |
| robots.txt | robots.txt | 放在網站根目錄，告訴爬蟲哪裡不能進的告示牌 |
| Sitemap | XML Sitemap | 你給 Google 的網站地圖，列出所有想被索引的頁面 |
| HTTPS | HTTPS | 安全加密的網址協定，網址列有鎖頭圖示 |
| 重定向 | Redirect | 把用戶/爬蟲從一個 URL 導向另一個 URL |
| 301 重定向 | 301 Redirect | 永久搬家的重定向，會傳遞 SEO 能量 |
| 302 重定向 | 302 Redirect | 暫時搬家的重定向，不傳遞 SEO 能量 |
| canonical | Canonical Tag | 告訴 Google 這個頁面的「正式版本」是哪個 URL |
| noindex | noindex | 告訴 Google 不要把這個頁面收錄進搜尋結果 |
| nofollow | nofollow | 告訴爬蟲不要追蹤這個連結（不傳遞 SEO 能量）|
| 快取 | Cache | 暫存的版本，Google 快取 = Google 上次讀到的頁面版本 |
| 渲染 | Rendering | 瀏覽器或爬蟲「執行」頁面程式碼後看到的結果 |
| 爬行預算 | Crawl Budget | Google 願意花在爬你網站的次數額度（大型網站才需要在意）|

## 頁面優化類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| 標題標籤 | Title Tag | 搜尋結果裡的藍色標題，也是瀏覽器分頁上的文字 |
| 描述標籤 | Meta Description | 搜尋結果裡 title 下方的灰色描述文字 |
| H1 | H1 (Heading 1) | 頁面最主要的標題，每頁只能有一個 |
| Alt Text | Alt Text / Alt Attribute | 圖片的替代文字，Google 靠它理解圖片內容 |
| 錨點文字 | Anchor Text | 超連結的可見文字（例如：點「這裡」的「這裡」）|
| 麵包屑 | Breadcrumb | 顯示頁面層級位置的導航（首頁 > 分類 > 文章）|
| On-Page SEO | On-Page SEO | 你能在頁面上直接控制的 SEO 元素 |
| Off-Page SEO | Off-Page SEO | 頁面外的 SEO 因素，主要是反向連結 |

---

## 連結類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| 反向連結 | Backlink / Inbound Link | 其他網站連結到你的連結 |
| 內部連結 | Internal Link | 你的網站內部頁面之間的連結 |
| 外部連結 | External Link | 從你的網站連出去到其他網站的連結 |
| 連結能量 | Link Equity / Link Juice | 連結傳遞的 SEO 排名能量 |
| 域名權重 | Domain Authority (DA) / Domain Rating (DR) | 第三方工具評估的網站整體連結權威分數（0-100）|
| 孤兒頁面 | Orphan Page | 沒有任何其他頁面連結指向的頁面 |
| 斷鏈 | Broken Link | 連結指向不存在的頁面（404 錯誤）|

## 搜尋結果類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| SERP | Search Engine Results Page | 搜尋引擎結果頁面（你輸入關鍵字後看到的整個頁面）|
| 精選摘要 | Featured Snippet | 搜尋結果最頂部的答案方塊，又叫 Position 0 |
| 豐富結果 | Rich Results / Rich Snippets | 有額外視覺元素的搜尋結果（星級評分、食譜資訊等）|
| 結構化資料 | Structured Data / Schema | 嵌入頁面的標準化資訊，幫助 Google 理解內容並解鎖豐富結果 |
| Knowledge Panel | Knowledge Panel | 搜尋結果右側的品牌/人物資訊卡片 |
| Local Pack | Local Pack | 搜尋本地服務時出現的地圖 + 三家商家列表 |
| PAA | People Also Ask | 搜尋結果中「其他人也問了」的問答展開框 |
| 點擊率 | CTR (Click-Through Rate) | 曝光次數中有多少比例被點擊（點擊 ÷ 曝光）|
| 曝光數 | Impressions | 你的頁面出現在搜尋結果的次數 |

## 內容類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| 關鍵字 | Keyword | 用戶在搜尋引擎輸入的詞語 |
| 長尾關鍵字 | Long-tail Keyword | 較長、較具體的搜尋詞，競爭低但意圖明確 |
| 搜尋量 | Search Volume | 這個關鍵字每月被搜尋幾次 |
| 搜尋意圖 | Search Intent | 用戶搜尋這個詞背後真正想達成的目的 |
| 關鍵字難度 | Keyword Difficulty (KD) | 在這個關鍵字排名的困難程度（0-100）|
| 主題地圖 | Topical Map | 系統性規劃網站要涵蓋哪些主題的架構圖 |
| 支柱頁面 | Pillar Page | 廣泛涵蓋某個大主題的核心頁面 |
| 群組頁面 | Cluster Page | 深入探討支柱主題下某個子主題的頁面 |
| 主題權威 | Topical Authority | Google 認定你的網站是某個領域的專家程度 |
| 長青內容 | Evergreen Content | 不受時間影響、長期有效的內容（vs 時效性內容）|
| 內容剪枝 | Content Pruning | 刪除或整合低品質、無流量的頁面 |
| 關鍵字蠶食 | Keyword Cannibalization | 自己的多個頁面互相競爭同一個關鍵字 |

## 指標類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| LCP | Largest Contentful Paint | 頁面最大內容元素的載入時間（Google 速度指標，目標 < 2.5 秒）|
| INP | Interaction to Next Paint | 互動後頁面反應的時間（目標 < 200ms）|
| CLS | Cumulative Layout Shift | 頁面載入時元素位移的程度（目標 < 0.1）|
| Core Web Vitals | Core Web Vitals | Google 的三個核心用戶體驗指標（LCP + INP + CLS）|
| GSC | Google Search Console | Google 的免費網站管理工具 |
| GA4 | Google Analytics 4 | Google 的免費網站流量分析工具 |
| E-E-A-T | Experience, Expertise, Authoritativeness, Trustworthiness | Google 評估內容品質的四個維度 |
| YMYL | Your Money Your Life | 影響財務/健康/安全的內容類型，Google 要求更嚴格 |

## 平台類

| 術語 | 英文 | 白話解釋 |
|------|------|---------|
| CMS | Content Management System | 內容管理系統，例如 WordPress、Shopify |
| GBP | Google Business Profile | Google 商家檔案（以前叫 Google My Business）|
| CDN | Content Delivery Network | 內容傳遞網路，讓用戶從最近的伺服器取得內容 |
| GTM | Google Tag Manager | Google 標籤管理工具，用來管理各種追蹤代碼 |
| SPA | Single Page Application | 單頁應用程式（React、Vue 等框架），SEO 需要特別注意 |
| SSR | Server-Side Rendering | 伺服器端渲染，SEO 最友好的 JavaScript 渲染方式 |
