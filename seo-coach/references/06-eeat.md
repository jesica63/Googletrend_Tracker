# E-E-A-T（專業度信號）

## 核心概念
E-E-A-T = **Experience（親身經驗）、Expertise（專業知識）、Authoritativeness（權威性）、Trustworthiness（可信度）**

這是 Google 品質評估人員（Quality Raters）使用的評估框架，特別對 **YMYL（Your Money Your Life）** 類型的網站影響最大。

給初學者的白話版本：
- 不要把 E-E-A-T 想成一個神祕分數
- 它比較像在問：**這個內容值不值得相信？這個人有沒有真的懂？**
- 對健康、財務、法律、醫療、購物決策這種會影響人生的重要主題，Google 會更在意
- 四個字裡面，**Trust（可信度）最重要**

**YMYL 包含：** 醫療健康、財務投資、法律建議、安全相關、購物決策等影響生活重要決策的內容。

---

## 四個維度詳解

### Experience（親身經驗）—— E-E-A-T 中最新的 E（2022年加入）
Google 開始重視作者是否有親身使用/體驗的經歷，而非只是書本知識。

**信號：**
- 文章中包含個人使用經驗、照片、案例
- 「我試過這個產品三個月」比「根據研究顯示」更有 Experience 信號
- 第一人稱的真實描述

### Expertise（專業知識）
作者或網站是否在這個領域具備足夠的知識？

**信號：**
- 作者 Bio（姓名、職稱、資歷、相關認證）
- 內容的深度和準確性
- 是否引用可信來源
- 是否有其他平台的專業發表（LinkedIn、學術論文等）

### Authoritativeness（權威性）
在這個領域，別人是否認可你為權威？

**信號：**
- 其他權威網站的反向連結
- 媒體報導、引用
- 品牌搜尋量（有人直接搜你的品牌名）
- 行業獎項、認證
- 社群媒體的追蹤數和互動

### Trustworthiness（可信度）—— 最重要的維度
Google 明確指出 Trustworthiness 是 E-E-A-T 的核心。

**基本信任信號：**
- 全站 HTTPS
- 清楚的聯絡方式（電話、Email、地址）
- 清楚的隱私政策和服務條款
- 退款政策（電商）
- 公司/個人的真實資訊（About 頁面）
- 用戶評價和評論（真實的）
- 無過多侵擾性廣告

給 Coach 的判斷問句：
- 這篇內容是誰寫的？
- 讀者憑什麼相信這個作者或這個品牌？
- 如果這頁在講重要決策，它有沒有把風險、來源、真實身份講清楚？

不要把 E-E-A-T 教成「加個 schema 就好」。Schema 可以幫助理解，但信任感本身不是靠標記假裝出來的。

## 怎麼提升 E-E-A-T 信號

### 作者頁面
每篇文章應有：
- 作者真實姓名
- 作者照片
- 作者 Bio（2-3 句說明專業背景）
- 連結到作者的完整介紹頁面

### About 頁面
- 公司/個人的真實故事
- 團隊成員介紹（有照片更好）
- 成立時間、地點
- 在業界的資歷或成就

### 初學者最容易先做的 4 件事
如果用戶是新手，不要一次丟完整 E-E-A-T 計畫。先從這四件事開始：

1. 補一個清楚的 About 頁
2. 每篇內容加作者名字和簡短介紹
3. 把聯絡方式、地址、Email、客服方式補完整
4. 檢查重要內容有沒有明確來源、實測經驗或真實案例

這四件事比空談「權威性」更容易開始。

### 聯絡資訊
- 至少一個聯絡方式（Email 最基本）
- 實體店面或公司 → 地址更重要
- 回應時間說明

### Schema Markup（強化 E-E-A-T 信號）
```json
{
  "@type": "Article",
  "author": {
    "@type": "Person",
    "name": "作者名稱",
    "url": "作者頁面網址",
    "sameAs": ["LinkedIn URL", "Twitter URL"]
  },
  "publisher": {
    "@type": "Organization",
    "name": "品牌名稱",
    "logo": {"@type": "ImageObject", "url": "logo圖片URL"}
  }
}
```

## E-E-A-T 的常見誤解

### 不是排名因素（直接的）
Google 沒有「E-E-A-T 分數」，它是品質評估人員使用的框架，通過演算法間接影響排名。

### 不是一套可以作弊的清單
不是補一個作者頁、插幾個認證 logo、加 schema 就會突然變可信。Google 更在意的是整體內容品質、真實身份、網站信任感、外部聲譽是否一致。

### 提升 E-E-A-T 需要時間
這不是設定好 Schema 就能立竿見影的事，需要持續產出高品質內容、累積外部信譽。

### 小網站也可以有高 E-E-A-T
E-E-A-T 不看網站規模，看的是在你專注的領域是否有深度和可信度。

### 有經驗不代表一定要「專家頭銜」
有些主題看重正式專業背景，有些主題則看重真實使用經驗。Google 在 2022 後更明確加入 Experience，就是因為有些內容最有價值的地方，來自真正做過、用過、踩過坑。

## Coach 教學提醒

- 新手先教 Trust，再教完整 E-E-A-T
- 先從「這頁有沒有讓人安心」開始，不要先講抽象縮寫
- YMYL 題材可以講得更嚴格；一般題材不用把 E-E-A-T 講得像審核制度
- 如果用戶問的是 Core Update 後掉流量，這裡只教基礎判斷；完整恢復策略屬專業範圍
